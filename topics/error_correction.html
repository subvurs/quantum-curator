QuantiSpect: A Structure-Aware Lightweight 3D CNN Pre-Decoder for Scalable Surface Code Quantum Error Correction

Real-time decoding is a critical bottleneck for large-scale fault-tolerant quantum computing. AI-based neural pre-decoders locally correct most physical errors before passing residual syndromes to a global decoder, enabling sub-microsecond latencies. However, existing architectures carry significant...

This article introduces QuantiSpect, a structure‑aware 3D CNN that trims the computational bulk of neural pre‑decoding for surface‑code error correction by exploiting separable spatial and temporal syndrome patterns, cutting parameters and MACs by roughly threefold while preserving the same receptive field as larger models. By delivering sub‑microsecond latencies and boosting logical error suppression up to 1.85× over uncorrelated PyMatching, it tackles a key bottleneck in real‑time fault‑tolerant pipelines that has limited recent AI‑decoder prototypes. The work dovetails with the broader push toward hardware‑friendly quantum‑error‑correction stacks, though full system integration will still require coupling to an efficient global decoder and validation on emerging quantum processors.
